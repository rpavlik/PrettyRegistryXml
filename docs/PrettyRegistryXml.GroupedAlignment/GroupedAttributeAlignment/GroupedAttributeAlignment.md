## Remarks

If you do not pass a "trailer" as your last element (an item for which [`IsTrailer`](../IAttributeSequenceItem/IsTrailer.md) is true, indicating it will consume all remaining attributes), one will be automatically added for you. It is an error to have one anywhere besides the last element.
